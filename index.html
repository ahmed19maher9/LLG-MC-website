Installation Instructions

1
Prerequisites
Ensure Java 21 is installed and JAVA_HOME is set. Install Maven for building from source.
2
Download
Clone the repository or download the latest release from GitHub.
3
Build
Run mvn clean compile to build the application.
4
Run
Execute mvn exec:java -Dexec.mainClass="vlcj.llg_mc.App"
5
Alternative
Build JAR with mvn clean package and run with java -jar target/llg-mc-0.0.1-SNAPSHOT.jar
